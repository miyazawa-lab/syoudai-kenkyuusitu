# 論文1
## 運動学的解析とスポーツフォームの測定

### 分かったこと
- スイングフォームを数値化することで、初心者と経験者の違いを客観的に比較できることが分かった
- 3次元ジャイロセンサを使うことで、ゴルフスイング中の体の回転や動きを定量的に計測できることが分かった
### 悩んだこと
- インパクト時の衝撃によって一部のセンサに誤差が生じる。
- 被験者が少なく、評価結果の一般性は十分ではない

# 論文2
## 運動過程をフィードバックする筋力トレーニング支援システム

### 分かったこと
- PoseNetで17個の関節座標を取得できる
- トレーニングを「開始・途中・終了」の段階に分類できる
- リアルタイムにフォームのフィードバックが可能
- 正しいフォームのみ回数をカウントできる

### 悩んだこと
- 被験者によって認識精度が異なる
- 動作速度が速いと誤認識が発生する
- PoseNetの精度に依存する
- フィードバックが文字表示のみである
