

# 今後の研究方針

## 研究概要

本研究では、筋力トレーニング（スクワットなど）を対象に、お手本動画と利用者のフォームを比較し、改善点をフィードバックするシステムの開発を目指す。

MediaPipeを用いて動画から姿勢を推定し、関節座標や関節角度を取得するところまで実装した。今後は、お手本動画との比較機能を追加し、フォームの違いを評価できるシステムへ発展させる。

---

## システムの流れ

```
動画入力
    ↓
姿勢推定（MediaPipe）
    ↓
関節座標の取得
    ↓
特徴量の抽出
（関節角度・動作時間・動作軌跡）
    ↓
お手本動画との比較
    ↓
フォーム評価
    ↓
改善点のフィードバック
```

---

## 今後取り組むこと

フォーム比較では、関節角度や動作時間、動作軌跡などの特徴量を利用し、お手本動画との違いを評価する。また、比較結果から問題点を検出し、改善点を分かりやすく表示できるようにする。

システムの基本的な流れは共通とし、競技ごとに評価項目のみを変更できる設計を考えている。

例

- 筋力トレーニング
- 関節角度 
- 可動域
- フォームの安定性

- 弓道
- 姿勢
- 動作のタイミング
- 「間」

このように、比較アルゴリズムは共通化し、競技ごとの特徴のみを変更することで、様々なスポーツへ応用できるシステムを目指す。
